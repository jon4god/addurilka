Плагин виджета для проверки индексации ссылки и добавления её в Яндекс.

== Description ==

Простой плагин, который добавляет на панель администратора виджет для проверки и добавления публикаций в поисковых системах.<br>
Можно посмотреть проиндексирована ли ссылка текущей страницы в ...<br>
...и добавить её в случае отсутствия в поисковые системы.

A simple plugin that adds a widget to the admin panel to add and verify the site links to search engine.

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.<br>
Установите плагин в Wordpress используя установщик плагинов.<br>
Активируйте плагин в вкладке Плагины.

Download plugin through the wordpress included plugin downloader.<br>
Activate plugin in plugin tab.
